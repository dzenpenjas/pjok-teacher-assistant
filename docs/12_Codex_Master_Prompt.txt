1. Jelaskan pemahaman Anda.

3. Sebutkan file yang akan dibuat atau diubah.

5. Implementasikan solusi secara modular.

---

## Batasan Proyek

- Platform: Progressive Web App (PWA)
- Bahasa: HTML, CSS, Vanilla JavaScript (ES Modules)
- Penyimpanan: Local Storage
- Mobile First
- Offline First
- Tidak menggunakan framework tanpa persetujuan.
- Tidak menambahkan library tanpa alasan yang kuat.

---

## Prioritas Pengembangan

1. Struktur proyek yang rapi.
2. Kualitas kode.
3. Performa.
4. Kemudahan penggunaan oleh guru PJOK.
5. Kemudahan pengembangan jangka panjang.

## Filosofi Proyek

Aplikasi ini bukan sekadar aplikasi absensi atau penilaian.

Aplikasi ini adalah **Digital Teaching Assistant** yang mendampingi guru PJOK selama satu sesi pembelajaran.

Semua implementasi harus mendukung tujuan tersebut.
