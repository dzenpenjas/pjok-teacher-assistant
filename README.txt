# PJOK Teacher Assistant

> Digital Teaching Assistant untuk Guru Pendidikan Jasmani, Olahraga, dan Kesehatan (PJOK).

## Tentang Proyek

PJOK Teacher Assistant adalah Progressive Web Application (PWA) yang dirancang khusus untuk membantu guru PJOK mengelola pembelajaran di lapangan.

Fokus utama aplikasi adalah mengurangi pekerjaan administrasi sehingga guru dapat lebih fokus mendampingi siswa.

---

## Tujuan

- Menggantikan buku nilai.
- Mempermudah absensi.
- Mempermudah penilaian praktik.
- Menyediakan stopwatch dan timer terintegrasi.
- Mendokumentasikan perkembangan siswa.
- Membantu identifikasi bakat olahraga.

---
